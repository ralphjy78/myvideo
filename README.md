# MyVideo

A modern React + Vite website with basic sections: Header, Hero, Features, Video, Contact, and Footer.

## Requirements
- Node.js 18+

## Get started
```bash
npm install
npm run dev
```
- Open the URL printed in the terminal (defaults to http://localhost:5173)

## Production build
```bash
npm run build
npm run preview
```
